Notes:

This HTML page attempts to use the quire.io API from the web browser client. It fails because of a CORS issue. The material returned from the quire.io api server is blocked from the browser.

So instead its necessary to have the server act as an intermediary. To do this, first use the URL
http://server/quireStart
This will require a page with a "Connect Quire" link. Click that link and it will bring up a page from quire asking about permissions. Grant permissions and it will bring up a callback page that should say "callback ok code ...."
Now the API can be used.
Commands like http://server/api/quire/getUser
http://server/api/quire/getOrganizations
